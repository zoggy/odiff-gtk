Odiff-gtk

What is Odiff-gtk ?

Odiff-gtk is a small OCaml library providing functions to display and merge diff results in a Gtk interface.

Odiff-gtk was previously part of OCamldiff but is now developped separately.

Download

Odiff-gtk is hosted on Github.

Releases

  • 1.0 [2013-06-03]: first release after extraction from OCamldiff.

Installation

Odiff-gtk requires LablGtk >= 2.14.1 and OCamldiff >= 1.1 installed with ocamlfind.

To compile and install:

  • tar xvfz odiff-gtk-XXX.tar.gz
  • cd odiff-gtk-XXX
  • make all install
